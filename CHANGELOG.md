# Changelog

## v1.1 — 2026-07-22

### 新增
- **搜索结果面板**：全局搜索改为在侧栏陈列所有匹配（歌名 / 歌手·专辑 双行条目），
  点击跳转到对应专辑、可连续试跳，`←` / `Esc` / 右键返回歌手列表（上限 400 条）
- **播放队列可视化**：加入播放队列的曲目行首显示琥珀序号角标（①②③…），随队列增删实时刷新
- **曲目右键菜单**：foobar2000 原生上下文菜单（移除 / 添加到播放队列 / 播放 /
  打开所在文件夹 / 标签 / 文件操作 / 转换 / 工具 / 属性）
- **封面墙专辑右键菜单**：原生菜单作用于整张专辑（整张加队列 / 转换 / 改标签等）
- **侧栏「正在播放」标记**：浏览其他歌手时，正在播放的歌手行保持淡琥珀底纹 + 圆点

### 改进
- 播放顺序由单键循环改为**三键并列**（顺序 / 列表循环 / 随机），当前模式琥珀标识
- 曲目行编码词（FLAC 等）降半调，码率数字成为技术列主角
- 侧栏歌手行间新增浅分割线
